Скрипт удаляющий в заданной дирректории все файлы старше указанного количества дней.

Переменные, в которые нужно подставить ваше значение в файле start.bat:
    DAYS - количество дней, старше которого файлы будут удаляться;
    FOLDER_PATH - путь к папке с файлами.

Второй вариант запустить файл start.bat из консоли указывая дни и пути, например:
.\start.bat 10  "C:\Кисляков\Скриншоты" "C:\Кисляков\Скриншоты-копия"
где 10 количество дней, старше которого файлы будут удаляться;
"C:\Кисляков\Скриншоты" "C:\Кисляков\Скриншоты-копия" - пути к папкам с файлами.

Скачайте файл с Github:
    Code - Download ZIP
Распакуйте архив.

Добавление скрипта в автозапуск Windows:
    1. Создайте ярлык bat-файла.
    2. Поместите ярлык в папку автозагрузки:
        Нажмите Win + R, введите:
        shell:startup
        Перетащите ярлык в открывшуюся папку.
Не требует прав администратора.
Работает только для текущего пользователя.

Для работы скрипта должен быть установлен python!

Проверить установку можно:
    Откройте командную строку (Win + R → cmd) и выполните:
        python --version
    Если команда не работает, попробуйте:
        py --version

Для установки Python:
    1. Скачивание Python
        Перейдите на официальный сайт Python.
        Выберите версию Python 3.10+ (рекомендуется последняя стабильная версия).
        Нажмите Download и скачайте установщик для вашей ОС (Windows/macOS/Linux).
    2. Установка (Windows)
            Запустите скачанный установщик (например, python-3.12.1-amd64.exe).
            Включите опции:
                ☑ Add Python to PATH (важно!)
                ☑ Install launcher for all users (рекомендуется)
            Нажмите Install Now.
            После установки закройте установщик кнопкой Close.

Проверьте установку в командной строке (Win + R → cmd) и выполните: python --version

Для Linux/macOS
Python обычно предустановлен. Проверьте версию:
    python3 --version

Если Python отсутствует, установите его через терминал:
    Ubuntu/Debian:
        sudo apt update && sudo apt install python3
    macOS (через Homebrew):
        brew install python
